# How To Run Locally

## Clone Repository

```bash
git clone https://github.com/Riktam45/personal-portfolio.git
```

---

## Open Project Folder

```bash
cd personal-portfolio
```

---

## Run Project

Simply open:

```text
index.html
```

in your browser.

OR use VS Code Live Server extension.

---

# VS Code Live Server (Recommended)

1. Open project in VS Code
2. Install:
   Live Server Extension
3. Right click:
   `index.html`
4. Click:
   `Open with Live Server`
